# E2E / System Tests Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Playwright E2E test suite that boots the real Docker Compose stack against a throwaway Postgres database and drives the full browser→API→DB path, with LLM-dependent specs gated behind an `OPENAI_API_KEY` env var.

**Architecture:** A top-level `e2e/` workspace (Playwright/TypeScript) runs against `http://localhost:5101` (nginx frontend container). A `docker-compose.e2e.yml` overlay adds healthchecks and a fixed JWT signing key so `--wait` is reliable and tokens survive process restarts. An `auth.setup.spec.ts` project runs once to create the first admin and persist `storageState.json`; all other specs depend on it.

**Tech Stack:** `@playwright/test` (latest), Docker Compose v2, GitHub Actions.

#   docker compose -f docker-compose.yml -f docker-compose.e2e.yml up --build -d --wait

# across container restarts, and uses ephemeral Postgres data (removed by `down -v`).

services:
  postgres:
    healthcheck:
      test: ["CMD-SHELL", "pg_isready -U proxytrace -d proxytrace"]
      interval: 5s
      timeout: 5s
      retries: 10

  api:
    depends_on:
      postgres:
        condition: service_healthy
    healthcheck:
      test: ["CMD-SHELL", "wget -qO- http://localhost:8080/api/health || exit 1"]
      interval: 5s
      timeout: 5s
      retries: 20
      start_period: 15s
    environment:
      - Authentication__Mode=Local
      - Authentication__Local__SigningKey=e2e-test-signing-key-32-chars-min!
      - Kiosk__Enabled=false

  proxy:
    depends_on:
      postgres:
        condition: service_healthy
      api:
        condition: service_healthy

  frontend:
    depends_on:
      api:
        condition: service_healthy

## Task 3: API helper — typed request wrappers

**Files:**
- Create: `e2e/helpers/api-client.ts`

- [ ] **Step 1: Write `e2e/helpers/api-client.ts`**

```typescript
import type { APIRequestContext } from '@playwright/test';

export interface AuthModeDto {
  mode: string;
  setupRequired: boolean;
  legacyClaimAvailable: boolean;
}

export interface TokenResponse {
  token: string;
  expiresAt: string;
}

export interface ProviderDto {
  id: string;
  name: string;
  endpoint: string;
  kind: string;
}

export interface ModelEndpointDto {
  id: string;
  providerId: string;
  name: string;
}

export interface ProjectDto {
  id: string;
  name: string;
  systemEndpointId: string;
}

export interface ApiKeyDto {
  id: string;
  name: string;
  value: string;
  projectId: string;
}

export interface SetupCompleteResponse {
  providerId: string;
  endpointId: string;
  projectId: string;
  apiKeyValue: string;
}

export class ProxytraceApiClient {
  constructor(
    private readonly request: APIRequestContext,
    private token?: string,
  ) {}

  private headers(): Record<string, string> {
    return this.token ? { Authorization: `Bearer ${this.token}` } : {};
  }

  async getAuthMode(): Promise<AuthModeDto> {
    const res = await this.request.get('/api/auth/mode');
    return res.json();
  }

  async setupAdmin(email: string, password: string): Promise<TokenResponse> {
    const res = await this.request.post('/api/auth/setup', {
      data: { email, password },
    });
    if (!res.ok()) throw new Error(`setup failed: ${res.status()} ${await res.text()}`);
    return res.json();
  }

  async login(email: string, password: string): Promise<TokenResponse> {
    const res = await this.request.post('/api/auth/login', {
      data: { email, password },
    });
    if (!res.ok()) throw new Error(`login failed: ${res.status()} ${await res.text()}`);
    return res.json();
  }

  setToken(token: string) {
    this.token = token;
  }

  async completeSetup(opts: {
    providerName: string;
    providerEndpoint: string;
    providerUpstreamApiKey: string;
    providerKind: string;
    modelName: string;
    projectName: string;
    apiKeyName: string;
  }): Promise<SetupCompleteResponse> {
    const res = await this.request.post('/api/setup/complete', {
      headers: this.headers(),
      data: {
        ...opts,
        inputTokenCost: null,
        outputTokenCost: null,
      },
    });
    if (!res.ok()) throw new Error(`setup/complete failed: ${res.status()} ${await res.text()}`);
    return res.json();
  }

  async getAgentCalls(params?: { page?: number; pageSize?: number }) {
    const qs = new URLSearchParams();
    if (params?.page != null) qs.set('page', String(params.page));
    if (params?.pageSize != null) qs.set('pageSize', String(params.pageSize));
    const query = qs.toString() ? `?${qs}` : '';
    const res = await this.request.get(`/api/agent-calls${query}`, {
      headers: this.headers(),
    });
    if (!res.ok()) throw new Error(`agent-calls failed: ${res.status()} ${await res.text()}`);
    return res.json();
  }
}
```

import { ProxytraceApiClient } from '../helpers/api-client';

const AUTH_FILE = '.auth/storageState.json';
const ADMIN_EMAIL = 'admin@e2e.test';
const ADMIN_PASSWORD = 'E2ePassword1!';

setup('create first admin and persist session', async ({ page, request }) => {
  const api = new ProxytraceApiClient(request);

  // Prove this is a fresh database — setup must be required.
  const mode = await api.getAuthMode();
  expect(mode.setupRequired, 'expected empty database — run `docker compose down -v` first').toBe(true);

  await page.goto('/');
  await page.evaluate((t) => localStorage.setItem('proxytrace.token', t), token);

  // Wait for the app to load (not the login page).
  await expect(page).not.toHaveURL(/\/login/);

  await page.context().storageState({ path: AUTH_FILE });
});

import { test, expect } from '@playwright/test';
import { ProxytraceApiClient } from '../helpers/api-client';

const PROXY_URL = 'http://localhost:5102';

test.describe('@llm ingestion via proxy', () => {
  test.skip(!process.env.OPENAI_API_KEY, 'requires OPENAI_API_KEY env var');

  let proxyApiKey: string;
  let authToken: string;

  test.beforeAll(async ({ request }) => {
    const api = new ProxytraceApiClient(request);
    const { token } = await api.login('admin@e2e.test', 'E2ePassword1!');
    authToken = token;
    api.setToken(token);

    // Re-run setup/complete to get a fresh key (idempotent in e2e).
    // If core-crud spec already ran, a key exists; just re-create another.
    const result = await api.completeSetup({
      providerName: 'E2E LLM Provider',
      providerEndpoint: 'https://api.openai.com/v1',
      providerUpstreamApiKey: process.env.OPENAI_API_KEY!,
      providerKind: 'OpenAi',
      modelName: 'gpt-4o-mini',
      projectName: 'E2E LLM Project',
      apiKeyName: 'e2e-llm-key',
    });
    proxyApiKey = result.apiKeyValue;
  });

  test('trace appears in UI after proxy request', async ({ page, request }) => {
    // Count existing agent calls before the request.
    const api = new ProxytraceApiClient(request);
    api.setToken(authToken);
    const before = await api.getAgentCalls({ pageSize: 1 });
    const countBefore: number = before.total ?? 0;

    // Send a real chat completion through the proxy.
    const proxyRes = await request.post(`${PROXY_URL}/openai/v1/chat/completions`, {
      headers: {
        Authorization: `Bearer ${proxyApiKey}`,
        'Content-Type': 'application/json',
      },
      data: {
        model: 'gpt-4o-mini',
        messages: [{ role: 'user', content: 'Reply with exactly: pong' }],
        max_tokens: 10,
      },
    });
    expect(proxyRes.ok(), `proxy returned ${proxyRes.status()}: ${await proxyRes.text()}`).toBeTruthy();

    // Poll until a new agent call appears (max 30 s).
    await expect.poll(
      async () => {
        const after = await api.getAgentCalls({ pageSize: 1 });
        return (after.total ?? 0) > countBefore;
      },
      { timeout: 30_000, intervals: [2_000] },
    ).toBe(true);

    // Verify the trace is visible in the Traces UI.
    await page.goto('/traces');
    await expect(page.getByText('gpt-4o-mini')).toBeVisible({ timeout: 10_000 });
  });
});
```

#!/usr/bin/env bash

# Running the E2E Test Suite

Proxytrace ships a Playwright end-to-end test suite in `e2e/` that boots the full production Docker Compose stack against a throwaway database and drives real browser interactions.

## Quick start

```bash
cd e2e
bash run.sh
```

This script:
1. Tears down any existing e2e stack and removes volumes (`docker compose down -v`).
2. Rebuilds and starts the stack (`docker compose up --build -d --wait`).
3. Runs all Playwright tests.
4. Tears the stack down again.

## With LLM specs

Pass an `OPENAI_API_KEY` to enable the gated `@llm` specs that exercise real proxy ingestion and test runs:

```bash
OPENAI_API_KEY=sk-... bash e2e/run.sh
```

Without the key, `@llm` specs are skipped automatically — the rest of the suite still passes.

## Compose ports (e2e overlay)

| Service  | Host port | Purpose |
|----------|-----------|---------|
| nginx (frontend) | 5101 | Playwright base URL |
| api | 5100 | REST / SSE |
| proxy | 5102 | OpenAI-compatible ingestion endpoint |
| postgres | 5432 | Database (fresh per run) |
| redis | 6379 | Ingestion transport |

## Running individual projects

```bash
cd e2e
npx playwright test --project=smoke          # Route smoke tests
npx playwright test --project=core           # CRUD verification
npx playwright test --project=llm            # LLM ingestion + test runs
```

## Viewing the report

```bash
cd e2e && npm run report
```

## Troubleshooting

**Stack does not become healthy:** Check `docker compose -f docker-compose.yml -f docker-compose.e2e.yml logs api` — the most common cause is a port conflict on 5101/5100/5102.

**`setupRequired` is `false`:** The Postgres volume was not removed. Run `docker compose -f docker-compose.yml -f docker-compose.e2e.yml down -v` before re-running.

**LLM specs time out:** The upstream OpenAI API may be rate-limited. Increase `timeout` in `playwright.config.ts` or retry.
```
