# Contributing

Proxytrace is a commercial product whose source is published for transparency under the
[Elastic License 2.0](LICENSE). To keep ownership of the codebase unambiguous, **we do
not accept pull requests** — external contributions are not merged, and open PRs will be
closed with a pointer to this policy.

What is very welcome:

- **Bug reports** — open a [GitHub issue](https://github.com/Proxytrace/Proxytrace/issues)
  with reproduction steps, the version you run, and logs where relevant.
- **Feature requests and feedback** — also a GitHub issue.
- **Security reports** — do **not** open a public issue; see [SECURITY.md](SECURITY.md).
